## вңЁ Why VidMax?

VidMax is built on a simple promise вҖ” **no ads, no tracking, no bloat.** Just a fast, beautiful media player for both video and music, that respects your device and your privacy. The primary engine is **Jetpack Media3 (ExoPlayer)** вҖ” Google's modern, battle-tested media framework вҖ” with **MPV** available as a powerful secondary engine for advanced users who need broader codec support.

---

## рҹҺ¬ Features

### Playback
- **Media3 (ExoPlayer) engine** вҖ” primary engine, smooth and battery-efficient
- **MPV engine** вҖ” secondary engine for advanced codec and format support
- **Music & Video** вҖ” full media player, not just video
- **Multiple player UIs** вҖ” Modern Player and Wavy Player styles
- **Subtitle support** вҖ” built-in subtitle parser for SRT and more
- **Speed control** вҖ” 0.25Г— to 2Г— playback speed
- **Aspect ratio control** вҖ” fit, fill, stretch and more
- **Shuffle & Repeat** вҖ” all modes including repeat-one

### Library
- **Folders view** вҖ” browse videos by folder, just like your file manager
- **Playlist support** вҖ” create and manage your own playlists
- **Audio mode** вҖ” play audio from video files with background playback
- **Smart search** вҖ” find any file in your library instantly
- **Sort & filter chips** вҖ” sort by name, date, size, duration

### UI & Experience
- **Material 3 design** вҖ” expressive motion, modern typography, tonal elevation
- **Mini player** вҖ” keep playing while browsing your library
- **28 handcrafted themes** вҖ” fully offline, no internet required, instant switching
- **Grid & List view** вҖ” switch layouts on the fly
- **No ads. Ever.** вҖ” completely free, always

## рҹҸ—пёҸ Architecture

VidMax follows a clean **MVVM architecture** with a unidirectional data flow:

```
UI Layer (Jetpack Compose)
    в”Ӯ
    в–ј
ViewModel Layer
    в”Ӯ
    в–ј
Repository Layer
    в”Ӯ
    в–ј
Data / Service Layer (MPV, MediaSession, ContentResolver)
```

**Tech Stack:**

| Layer | Technology |
|---|---|
| Language | Kotlin 100% |
| UI | Jetpack Compose + Material 3 |
| Architecture | MVVM + Clean Architecture |
| Primary Engine | Jetpack Media3 (ExoPlayer) |
| Secondary Engine | MPV (via JNI bridge) |
| Audio Session | Android MediaSession / AudioService |
| Build System | Gradle (Kotlin DSL) |
| CI/CD | GitHub Actions |

## рҹҡҖ Getting Started

### Requirements
- Android Studio Hedgehog or newer
- Android SDK 21+
- NDK (for MPV native library)

### Build

```bash
# Clone the repository
git clone https://github.com/HATAKE2008/vidamx.git
cd vidamx

# Build debug APK
./gradlew assembleDebug
```

Or download the latest APK from [**Releases**](https://github.com/HATAKE2008/vidamx/releases).
