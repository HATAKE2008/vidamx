## ‚ú® Why VidMax?

VidMax is built on a simple promise ‚ÄĒ **no ads, no tracking, no bloat.** Just a fast, beautiful video player that respects your device and your privacy. Under the hood runs the battle-tested **MPV engine**, one of the most capable media players ever built, wrapped in a clean Material 3 interface crafted entirely in Kotlin.

---

## ūüé¨ Features

### Playback
- **MPV-powered engine** ‚ÄĒ hardware-accelerated, codec-rich, format-versatile
- **Multiple player UIs** ‚ÄĒ choose between Modern Player and Wavy Player styles
- **Subtitle support** ‚ÄĒ built-in subtitle parser for SRT and more
- **Speed control** ‚ÄĒ 0.25√ó to 2√ó playback speed
- **Aspect ratio control** ‚ÄĒ fit, fill, stretch and more
- **Shuffle & Repeat** ‚ÄĒ all modes including repeat-one

### Library
- **Folders view** ‚ÄĒ browse videos by folder, just like your file manager
- **Playlist support** ‚ÄĒ create and manage your own playlists
- **Audio mode** ‚ÄĒ play audio from video files with background playback
- **Smart search** ‚ÄĒ find any file in your library instantly
- **Sort & filter chips** ‚ÄĒ sort by name, date, size, duration

### UI & Experience
- **Material 3 design** ‚ÄĒ expressive motion, modern typography, tonal elevation
- **Mini player** ‚ÄĒ keep playing while browsing your library
- **28 handcrafted themes** ‚ÄĒ fully offline, no internet required, instant switching
- **Grid & List view** ‚ÄĒ switch layouts on the fly
- **No ads. Ever.** ‚ÄĒ completely free, always

| Layer | Technology |
|---|---|

## ūüöÄ Getting Started

### Requirements
- Android Studio Hedgehog or newer
- Android SDK 21+
- NDK (for MPV native library)

### Build

```bash
# Clone the repository
git clone https://github.com/HATAKE2008/VidMax.git
cd VidMax

# Build debug APK
./gradlew assembleDebug
```

Or download the latest APK from [**Releases**](https://github.com/HATAKE2008/VidMax/releases).
